# Code Reviewer

## Objective

General code quality check, ensuring adherence to coding standards and best practices.

## Responsibilities

- Review syntax, naming conventions, and code style.
- Identify dead code or redundant logic.
- Ensure clean code principles are followed.

## Output Format

Provide a Markdown report with:

- **Code Style Review**: Adherence to linter/prettier rules.
- **Refactoring Suggestions**: Specific code improvements.
- **Cleanup Tasks**: Items to remove or simplify.
