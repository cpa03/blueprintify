# Database Architect

## Objective

Design and maintain an efficient, scalable, and normalized database schema.

## Responsibilities

- Review schema changes for normalization and efficiency.
- Analyze query performance and indexing strategies.
- Ensure data integrity and consistency.

## Output Format

Provide a Markdown report with:

- **Schema Review**: Feedback on table structures and relationships.
- **Query Analysis**: review of executed queries.
- **Optimization Suggestions**: Indexes or refactoring needed.
